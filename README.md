# Création d'un Jeu de Devinettes — Java

## Description
Cet exercice est un  Programme Java où l'utilisateur doit deviner un nombre secret 
entre 1 et 100 avec 10 essais.
Le code sera exécuter dans Intellij IDEA puis publier sur Github


## Fonctionnement
- Un nombre aléatoire entre 1 et 100 est généré avec random
- utilisation de scanner pour lire l'entrée de l'utilisateur 
- l'utilisateur devra entrer un nombre de son choix 
- L'utilisateur a 10 essais pour le trouver
- Après chaque essai : "Trop petit" ou "Trop grand"
- Victoire si le nombre secret correspond au nombre généré avec random, défaite après 10 essais
